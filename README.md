## 📚 Course Overview
Full Stack Open 2025 is a modern web development course focusing on JavaScript-based technologies. The course covers:
- React
- Redux
- Node.js
- Express
- MongoDB
- GraphQL
- TypeScript
- Containerization with Docker
- CI/CD and testing
